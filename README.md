## Modulo Three

This little application gives several options for finding the remainder of division by 3.


#### To run it localy

1. Clone GitHub repo [Modulo Three](git@github.com:n0ne/mod-3-fsm.git)
2. `cd mod-3-fsm`
3. `npm install`
4. `npm run test:all`
